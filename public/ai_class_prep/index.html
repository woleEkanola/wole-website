What is OpenCode?

Imagine having a master software developer sitting inside your computer. You don't need to learn a programming language to talk to them; you just explain what you want in plain English.

OpenCode is an AI-powered desktop application that does exactly this. It listens to your everyday language and writes the complicated computer code behind the scenes to build websites, apps, and tools for you. It takes away the heavy lifting so you can focus entirely on your creativity.

Part 1: How to Install OpenCode

Step 1: Go to the website

Click the button below to open the official download page in a new tab:

Go to OpenCode Downloads
Step 2: Choose the right version for your computer

Scroll down to the [2] OpenCode Desktop (Beta) section. You will see a few options. Click "Download" next to the one that matches your computer:

  • Using a Windows PC? Choose Windows (x64).
  • Using an older Mac? (Bought before late 2020) Choose macOS (Intel).
  • Using a newer Mac? (With an M1, M2, or M3 chip inside) Choose macOS (Apple Silicon).
    Tip: If you aren't sure, click the Apple logo in the top left corner of your screen, select "About This Mac", and look at the "Chip" or "Processor" line.
Step 3: Install the app

Once the file downloads, double-click it. Follow the normal steps you usually take to install a program on your computer (like clicking "Next" or dragging the icon to your Applications folder).
